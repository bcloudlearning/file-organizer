# Variables

A variable is a named container that stores data.

Examples:

name = "Bharath"
age = 30

Variables allow programs to reuse data instead of hardcoding values everywhere.


========================

# Input

input() allows a program to receive data from the user while the program is running.

Example:

name = input("Enter your name: ")

The value entered by the user is stored in a variable.

==========================

# Conditions

Conditions allow a program to make decisions.

Example:

if age >= 18:
    print("Adult")
else:
    print("Minor")

Conditions are the foundation of business logic.

=================================
# Loops

Loops allow us to execute the same block of code multiple times.

Example:

for i in range(5):
    print("Hello")

Loops are useful for automation and repetitive tasks.

================================

# Functions

Functions are reusable blocks of code.

A function can:

- Accept inputs (parameters)
- Perform actions
- Be called multiple times

Example:

def greet(name):
    print(f"Hello {name}")


=================================
# Lists

Lists allow multiple values to be stored in a single variable.

Example:

clouds = ["AWS", "Azure", "GCP"]

Common operations:

- Access items using index
- Add items using append()
- Get length using len()
- Loop through items
